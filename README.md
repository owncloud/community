# 💬 ownCloud Community

Welcome to the central discussion hub for the ownCloud organization.

This repository hosts our **GitHub Discussions**, where users, contributors, and maintainers can connect, ask questions, share ideas, and collaborate beyond code.

---

## 🚀 What is this for?

Use Discussions here for:

* 🙋 **Help & Support** — Ask questions and get help from the community
* 💡 **Ideas & Feature Requests** — Share suggestions and improvements
* 💬 **General Discussion** — Talk about use cases, workflows, or anything related
* 🧪 **Show & Tell** — Share projects, integrations, and experiences
* 📣 **Announcements** — Official updates from maintainers

---

## ❗ What NOT to use this for

Please **do not** use Discussions for:

* 🐛 Bug reports → Use the relevant repository’s *Issues*
* 🔧 Specific feature implementation tasks → Use *Issues*
* 🔒 Security vulnerabilities → Follow responsible disclosure guidelines

---

## 🔗 Where to go instead

* Issues & bugs → Open an Issue in the relevant repository
* Documentation → Check the docs in each project repo
* Security concerns → Contact maintainers privately

---

## 🧭 Guidelines

To keep things useful and welcoming:

* Be respectful and constructive
* Search before posting to avoid duplicates
* Use the appropriate category
* Mark answers when your question is resolved

---

## ❤️ Contributing

Everyone is welcome to participate—whether you're asking questions, helping others, or sharing ideas.

If you’d like to contribute to the codebase, please check the individual repositories for contribution guidelines.

---

## 📌 Getting started

👉 Head over to the **Discussions** tab and join the conversation!
